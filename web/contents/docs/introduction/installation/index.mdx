To install and use readmit for your projects, follow this steps:

## Prerequisites (GO)

<Step>
<StepItem title="System Requirements">

- **Go:** Make sure you have Go installed. The project uses `go`. Install [Go](https://go.dev) here.

To verify installation, run:

```bash
go
```

</StepItem>
<StepItem title="readmit">
To install readmit, run this command:

```bash
go install github.com/treasureuzoma/readmit/readmit@latest
```

</StepItem>

</Step>
