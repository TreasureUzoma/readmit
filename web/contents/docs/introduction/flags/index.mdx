## Available Flags

### `generate docs` Flags

#### `--single-file`

- **Description**: Generates all documentation into a single `docs.md` file rather than creating a `docs/` folder with multiple files.
- **Example**:
  ```bash
  readmit generate docs --single-file
  ```
