## What is readmit?

**Readmit** is a simple CLI tool to generate beautiful READMEs, DOCS or guides for your projects.

Readmit is proudly built with Go.

## Key Features

**Readmit** offers a wide range of features to empower your analytics journey:

| Feature                        | Description                                                                      |
| ------------------------------ | -------------------------------------------------------------------------------- |
| **Readme generating**          | Generates a `README.md` file in the current directory.                           |
| **CONTRIBUTION.md generating** | Generates a `CONTRIBUTING.md` file in the current directory.                     |
| **App Documentation**          | Generates comprehensive app and API docs in a folder or single file.             |
| **Commit messages**            | Generates a commit message (optimized: only sends diffs).                        |
| **Vulnerability Report**       | Generates a `REPORT.md` file containing all codebase vulnerabilities with fixes. |
| **.gitignore Support**         | Respects project `.gitignore` rules and safely skips ignored files.              |
| **Smart Push**                 | One command to stage, AI-generate commit, commit, and push.                      |
| **Command Suggestions**        | Automatically suggests the closest valid command if you mistype.                 |
